# IR Remote Arduino

just simple app to control arduino (using ir receiver).

## Sketch

|  Component  |     Arduino Pin     |
|-------------|---------------------|
| LED         | PIN 13 (+), GND (-) |
| IR RECEIVER | PIN 12, GND, 5V     |
